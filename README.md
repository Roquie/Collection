# Collection
Simple PHP Collection based on Laravel 5 Collection.

## Install

`composer require roquie/collection`

## Usage (in progress ...)

Five minutes example: <br>
```php
$array = [
    'foo' => [
        'one',
        'two',
        'three'
    ],
    'bar' => [
        'baz' => [
           1, 2, 3, 4
        ],
        'elements' => [
          // etc...
        ]
    ]
];

```
